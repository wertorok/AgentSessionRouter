The verifier prompt at `src/cluster.ts:498-511` instructs Claude:

> *"Do not infer. Do not use tools. Return JSON only."*

The verifier is intentionally a **no-tools, no-inference** task — it just checks whether evidence files support the claims in the factsheet. Running it in `agent` mode would allow tool use and exploration, which defeats the purpose: the verifier should read only the provided evidence, not discover new code. `bare` (or `focused` as fallback) enforces this constraint at the CLI level.
