The LLM verifier is run in `bare` mode (or `focused` no-tools mode) and not `agent` mode because it must judge only the supplied evidence — not infer beyond it, not perform new discovery, and not use tools. It is meant to return `VERIFIED` only when the claim directly follows from the evidence provided, and return JSON only. Using `agent` mode would allow the model to explore and infer, which would violate the semantic verification contract.
